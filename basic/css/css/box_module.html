盒状模型

这是一个消去了margin和padding的容器

首先看看border

  • 边框可以有各种样式
    • double
    • dashed
    • groove
    • dotted
    • inset
    • outset
    • ridge
  • 边框还可以有宽度
  • 边框当然可以有颜色
  • 还可以对单个边框进行设置（一共上下左右四个边框）
  • 你也可以通过border-style一次性设置四个边框或者更少的边框

边框的简写使用border，其后分别是粗细，样式和颜色

再来看看Outline属性，就是边框最外面那个线

  • 首先是颜色属性
  • 然后是样式，这个和border一样
  • 然后是粗细
  • 最后是偏移量

简写是outline:width style color

margin

margin 清除周围的（外边框）元素区域。margin 没有背景颜色，是完全透明的。

也就是说，margin本身就是代表长度，所以margin属性都是给出长度

这里margin=0px

这里margin=100px

padding

当元素的 padding（填充）内边距被清除时，所释放的区域将会受到元素背景颜色的填充。

所以padding和margin一样，都是直接给出长度

padding=20px
