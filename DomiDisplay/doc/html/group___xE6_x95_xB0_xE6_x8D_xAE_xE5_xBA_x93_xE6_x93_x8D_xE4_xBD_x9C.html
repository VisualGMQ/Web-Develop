def initDomits.searchDB (   db,
  table,

  fetch = -1,

数据库查找函数

参数
condition给出查找条件，条件中不需要加上WHERE，会自动将这个参数和WHERE语句拼接
fetch决定cursor获得多少数据：
  • 1：执行.fetchone()获取一条数据
  • >1：执行.fetchmany()获取多条数据
  • <1：执行.fetchall()获取所有数据 默认为-1
table要查找的表名
cols要匹配的列名，默认为全部匹配
OrderBy是否按序查找，默认不按序查找。可以传入按序的列名（只能按照一列）
isDesc在按序查找的情况下是否降序查找，默认为False
返回
返回查询结果，使用元组返回

如果db为None，那么函数会通过你给出的参数来进行一次性查询（在内部打开和关闭数据库）。 如果db为已经生成的数据库链接，那么查询之后不会关闭数据库。
