函数的声明很简单，你只需要使用function关键字声明一个函数即可，而且参数不需要给出类型,不需要返回值，像是这样:
function sum(a,b){ return a+b; }
你也可以通过return;来返回空值 

对象的创建有三种形式:
	1. 使用构造函数，这种形式有点像python：
		function person(name,age){
			this.name=name;
			this.age=age;
			this.getName=function(){
				return this.name;
			}
		}
		然后使用new来创建对象：
		var p=new person("haha",18);
	2.使用替代语法，这种语法的语法格式很像Json：
		var p={
			name:"haha",
			age:18,
			getName:function(){
				return name;
			}
		};
		但是这种语法只能声明一个对象，是临时的。
	3.创建一个Object对象（Object是所有对象的基类），然后动态添加属性和方法：
		var p=new Object();
		p.name="haha"
		p.age=17;
		p.getName()=function(){
			return name;
		};
		这也是临时的。
	
使用对象的话，你可以通过.运算符来访问属性和方法，也可以像C++里面的字典一样访问属性：
p["name"];
	如果你对对象方法这样访问的话：
	p.getName;
	会像属性一样打印出这个函数的内部代码。所以你要是想正确使用成员函数的话，不要忘记加上括号
